## Players globales

| Empresa | Tipo | Fortaleza en AI | Oportunidad para Globant |
|---------|------|-----------------|--------------------------|

## Tendencias 2025-2026

1. **LLM-powered NPCs** — Los NPCs con diálogo generativo están llegando a todos los estudios
2. **AI-assisted game dev** — godot-ai, Cursor + Godot, GitHub Copilot para game code
3. **Procedural generation con diffusion** — Mundos, texturas, música generados por AI
4. **RL para game balance** — Entrenar agentes para testear y balancear juegos automáticamente
5. **Anti-cheat con AI** — Detección de comportamiento anómalo con ML
6. **GameCoder LLMs** — Modelos especializados en código de juegos (GameCoder-27B)

## Oportunidades para Globant en LATAM

- **Estudios indie** que no pueden pagar Unity Pro ni Unreal licensing → Godot + AI services
- **Serious games** para educación/salud → Godot + LLM NPCs personalizados
- **Game analytics** para estudios medianos → stack PostHog/Grafana + AI insights
- **Modernización de juegos legacy** → añadir AI a juegos existentes sin reescribir
- **Game testing AI** → reduce QA manual un 70% en proyectos de clientes
